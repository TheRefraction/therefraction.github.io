TROUBLESHOOTING


# The game failed to launch and could not initialize D3D Surfaces.

  • Either there aren't enough graphic resources, or your video card is not supported.

    • Try:

         - Closing everything else.

         - Shrinking resolution.

         - Make sure all video card drivers are up to date and the latest version of DirectX is installed.


    # A running cat appeared on screen.

  • Either you tried to load an invalid map or the save file is corrupt.

    • Try:

         - Restarting the game.

         - Starting a new game.

         - Deleting your save file.


    # Cannot open AppImage file on Linux.

    Try the following:

         - Install the package libfuse2 or enter "sudo apt install libfuse2" in the terminal (to do once).

         - Download the DESTORY AppImage file.

         - Authorize its execution (right-click on the file -> Properties -> Permissions -> Allow executing file as program).

         - Double-click on the file to open DESTORY.


    CONTACT

    If the troubleshooting section didn't help you, email technical support.

    To do it properly, please write down:


  • The operating system you're running on
  • Your computer's hardware configuration (better, but not necessary)
  • The game's version
  • A complete and detailed description of the bug (how it occurred? ; what did it do? ; ...)
  • And screenshots (if possible)

    • Finally, send an e-mail to:

    gmail.com

    Fixes and bugs patchs will be mentioned in changelogs.
